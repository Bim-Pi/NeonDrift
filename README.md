# NEON DRIFT — DeltaHacks 


A fast, juicy, arcade runner built with **pure HTML/CSS/JS (Canvas)** — no libraries, no assets.
This project was created in collaboration with MangoKrish

## How to play
- **Move:** Arrow keys (←/→) or **A/D**
- **Boost:** **Space** (uses Energy)
- **Pause:** **P**
- **Restart:** **R**
- **Help:** **H**
- **Mute:** **M**

**Mobile**
- Drag left/right to move
- **Two-finger tap** to boost

## Goal
Survive as long as possible, collect **Energy Orbs**, maintain your **Chain multiplier**, and grab powerups:
- **Shield (S):** blocks 1 hit
- **Magnet (M):** pulls orbs toward you
- **Slow-mo (⏱):** slows hazards briefly

## Run locally
Just open `index.html` in a browser.

Tip: if your browser blocks audio, click **Play** once to enable it.

## Deploy on GitHub Pages
1. Create a new GitHub repo (public is easiest).
2. Upload these files: `index.html`, `style.css`, `game.js` (and this README if you want).
3. Go to **Settings → Pages**
4. Under **Build and deployment**, select:
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
5. Save. Your link will appear (usually `https://<username>.github.io/<repo>/`).

Good luck at DeltaHacks!
